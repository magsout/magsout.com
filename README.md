# [magsout.com](https://magsout.com/)

magsout personal website.

## Installation

```ssh
git clone git@github.com:magsout/magsout.github.io.git
cd magsout.github.io
npm install
```

## Development

```ssh
npm run dev
```

## Static build

```console
npm run build
```

## lint

```console
npm run lint
```
