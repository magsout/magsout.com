# [magsout.com](https://magsout.com/)

magsout personal website.

## Installation

```console
git clone git@github.com:magsout/magsout.com.git
cd magsout.com
npm install
```

## Development

```console
npm run dev
```

## Static build

```console
npm run build
```

## lint

```console
npm run lint
```
